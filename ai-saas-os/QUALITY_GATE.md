# 版本质量门禁

任何版本标记 STABLE 前，必须通过以下检查。

基础检查：
1. composer install 成功
2. php artisan migrate:fresh --env=testing 成功
3. php artisan test 成功
4. composer audit 没有未解决高危漏洞
5. 不存在未处理异常
6. 不存在调试输出
7. 不存在无用测试假通过

v0.1.x 核心版本稳定标准：
1. 用户注册可用
2. 用户登录可用
3. 用户退出可用
4. License 创建可用
5. License 校验可用
6. 订单创建可用
7. 支付回调可用
8. 支付成功后可自动开通授权
9. CHANGELOG.md 已更新
10. STABLE_TAG.md 已标记当前版本

v0.2.x AI计费版本稳定标准：
1. 余额账户可用
2. 额度发放可用
3. 额度扣减可用
4. 使用记录可用
5. AI请求必须先检查License
6. AI请求必须先检查余额
7. 扣费必须走账本
8. 测试覆盖正常扣费和余额不足

v0.3.x 插件版本稳定标准：
1. 插件上传可用
2. 插件版本管理可用
3. 插件下载需要授权
4. 插件安装记录可用
5. 插件更新检查可用

v0.4.x 工作流版本稳定标准：
1. 事件触发可用
2. 条件判断可用
3. 动作执行可用
4. 执行日志可用
5. 失败可重试

v0.5.x 风控版本稳定标准：
1. 黑名单可用
2. 接口限流可用
3. 异常License记录可用
4. 风险事件可用
5. 高风险操作有记录

v0.6.x 营销渠道版本稳定标准：
1. 推广链接可用
2. 分销关系可用
3. 佣金记录可用
4. 续费提醒可用
5. 自动通知可用
